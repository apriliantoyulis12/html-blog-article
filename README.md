# Local Dev Journal

A personal coding journal project created to practice text formatting and heading hierarchy in HTML as part of The Odin Project curriculum.

## 📝 What I Practiced

- **Headings:** Organizing content structurally using `<h1>`, `<h2>`, and `<h3>` tags.
- **Text Formatting:** Emphasizing important text using `<strong>` (bold) and `<em>` (italic) tags.
- **Paragraphs:** Structuring readable article content using `<p>` tags.

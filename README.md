# Coding Challenge: Basic HTML Blog Article Page

This project is the solution to a coding challenge focused on creating a simple blog article structure. The main objective is to solidify the foundational understanding of text hierarchy and formatting elements in HTML, without using any CSS or JavaScript.

## 🎯 Challenge Constraints

As per the challenge rules, this page is built strictly using ONLY the following HTML tags:

- <h1> to <h3> (Headings for title structure)
- <p> (Paragraphs for body text)
- <strong> (To bold important text)
- <em> (To italicize emphasized text)

## 📂 File Structure

- index.html - The main file containing the entire structural code of the blog article.

## 🚀 How to Run the Project

1. Clone this repository to your local machine:
   git clone https://github.com/apriliantoyulis12/html-blog-article.git
2. Open the project folder.
3. Double-click the `index.html` file to view the result directly in your preferred web browser.
